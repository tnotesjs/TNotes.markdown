## 2. 🫧 评价

markdown 实战部分的建议：

- 参考菜鸟教程上的“个人简历制作示例”来实战体验
  - 把教程中的 markdown 语法都瞅瞅，看看是否有不理解的语法，查漏补缺
  - 也可以自己尝试用 markdown 写一份个人简历
  - 备注：就个人而言，从第一份工作起，简历都是使用 markdown 写的
- 随便找几篇以往写过的一些笔记或者文章，使用 markdown 来重写一遍
