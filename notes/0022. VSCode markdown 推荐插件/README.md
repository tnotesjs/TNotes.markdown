## 2. 🫧 评价

如果你也使用 VSCode 来书写 markdown 文档，那么可以考虑安装以下插件：

| 插件 | 简介 |
| --- | --- |
| [markdownlint][1] | Markdown 语法检查插件，当文档中出现不规范的写法时，该插件会实时提醒你，帮助保持文档风格一致。 |
| [Markdown All in One][2] | 功能全面的 markdown 增强插件，提供快捷键、自动生成目录、实时预览、格式化等一站式支持。 |
| [Markdown Preview Enhanced][3] | 功能强大的预览插件，支持数学公式、Mermaid 图表、代码块高亮、导出 PDF/HTML 等多种扩展特性。 |
| [Paste Image][4] | 支持从剪贴板直接粘贴图片到 markdown 文档，自动保存图片文件并插入对应的图片链接。 |
| [Markdown PDF][5] | 将 markdown 文档一键导出为 PDF、HTML、PNG 等格式，方便分享与归档。 |
| [Markdown Mermaid][6] | 为 VSCode 内置的 markdown 预览添加 Mermaid 图表渲染支持。 |

更多插件，可以在 VSCode 的插件市场中搜索 `markdown` 关键词，找到适合自己需求的插件来提升 markdown 的书写效率和体验。
