- github.dev 简介

## 2. 🫧 评价

如果要快速编辑 github 仓库中的一些内容，可以尝试使用 github.dev 来快速打开一个在线的编辑器。

做法很简单，在 github 仓库中，按下 `.` 键，或者将网址中的 `.com` 替换为 `.dev`，就可以进入 github.dev 的在线编辑器环境了。

## 3. 🔍 查看 github.dev 的简介

打开 [githbu.dev][2] 之后，会看到以下内容：

![img](https://cdn.jsdelivr.net/gh/tnotesjs/imgs-2026@main/2026-03-02-20-12-44.png)

中文翻译如下：

### 3.1. 这是什么？

github.dev 网页版编辑器是一种轻量级的编辑体验，完全在浏览器中运行。您可以浏览 GitHub 上的文件和源代码仓库，并进行代码修改和提交。

有两种方式可以直接在浏览器中进入 VS Code 环境开始编码：

- 在任何仓库或拉取请求页面按下 `.` 键。
- 将网址中的 `.com` 替换为 `.dev`。例如，仓库 https://github.com/github/dev 可以变为 http://github.dev/github/dev

预览下面的动图，快速了解 github.dev 的实际操作演示。

![github dev](https://user-images.githubusercontent.com/856858/130119109-4769f2d7-9027-4bc4-a38c-10f297499e8f.gif)

### 3.2. 为什么使用它？

这是一种编辑和浏览代码的快捷方式。如果您需要同时编辑多个文件，或者在进行快速更改时希望利用 Visual Studio Code 所有强大的代码编辑功能，它尤其有用。更多信息，请参阅我们的[文档](https://github.co/codespaces-editor-help)。
